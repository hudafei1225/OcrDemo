前言
        最近项目要增加识别身份证和二维码的需求，大概在网上搜了一下，没有合适的开源库。有一个开源的叫 tesseract 的扫描库，下载demo运行后的效果并不理想，识别耗费的时间过长，而且识别度不怎么高。随又参考别人的博客，接入了百度文字识别的API，但是网上搜出来的大多是直接调用百度文字识别已有的功能，护照的识别百度并未集成，这里需要自定义模板，稍后会写到具体自定义的方法。
